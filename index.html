Spam Gen es un simulador técnico de envío masivo de emails (spam) desarrollado con Node.js y Nodemailer, creado exclusivamente para fines académicos. Su objetivo es practicar conceptos como:

  • Diseño y creación de la web usando teconoligias HTML + CSS
  • Implementación de JavaScript usando JQUERY
  • Uso de la libreria nodemailer en JavaScript

Este sistema no debe utilizarse para enviar correos reales no solicitados. Violaría leyes anti-spam (como la CAN-SPAM Act o GDPR) y políticas de proveedores de email (Gmail, Outlook, etc...). El código está limitado a entornos controlados (localhost o cuentas de prueba) para evitar usos maliciosos.
